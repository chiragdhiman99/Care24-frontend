## рҹҢҹ What is Care24?

> **Care24** isn't just a platform вҖ” it's a bridge between vulnerability and compassion.

Imagine your elderly parent needs a trained nurse at home. Or a post-surgery patient needs daily wound care. Or a new mother needs a verified caregiver. **Care24 makes that happen in minutes.**

A full-stack home healthcare booking platform where patients can:
- рҹ”Қ **Discover** verified caregivers filtered by specialization, rating & location
- рҹ“… **Book instantly** with real-time slot selection
- рҹ’і **Pay securely** via Razorpay with instant confirmation
- рҹ“§ **Get notified** automatically via email the moment a booking is confirmed

## рҹҡҖ Getting Started вҖ” Run Locally in 60 Seconds

### Prerequisites

Make sure you have these installed:

| Tool | Version |
|------|---------|
| Node.js | v18+ |
| npm / yarn | Latest |
| Razorpay Account | Test Mode |

### вҡЎ Quick Setup

```bash
# 1пёҸвғЈ  Clone the repository
git clone https://github.com/chiragdhiman99/Care24-frontend.git

# 2пёҸвғЈ  Jump into the project
cd Care24-frontend

# 3пёҸвғЈ  Install all dependencies
npm install

# 4пёҸвғЈ  Set up environment variables (see below)
cp .env.example .env

# 5пёҸвғЈ  Fire it up рҹ”Ҙ
npm run dev
```

> рҹҹў App will be running at **http://localhost:5173**

## рҹ”— API Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/auth/register` | Register a new user |
| `POST` | `/api/auth/login` | Login & receive JWT |

| `POST` | `/api/payment/create-order` | Create Razorpay order |
| `POST` | `/api/payment/verify-payment` | Verify signature & confirm booking |

## рҹҢҗ Deployment

The frontend lives on **Vercel** with auto-deployments triggered on every push to `main`.

```bash
# Build for production
npm run build

# Preview production build locally
npm run preview
```

The `vercel.json` handles SPA routing вҖ” all paths redirect to `index.html` so React Router works correctly on Vercel.

```json
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/" }
  ]
}
```
